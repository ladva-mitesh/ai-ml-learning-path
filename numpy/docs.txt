---------------------------------------------------------------------------------------------
                                    CONCEPTS SUMMARY
---------------------------------------------------------------------------------------------

-> import numpy as np                       : standard way to import the numpy library
-> __version__                              : check the installed version of numpy
-> np.arange                               : create a 1D array of evenly spaced values
-> np.full                                 : create an array filled with a specified constant value
-> np.reshape                              : change the shape of an array without altering its data
-> array.copy()                            : create an independent copy so the original stays unchanged
-> np.vstack                               : stack arrays vertically (row-wise)
-> np.hstack                               : stack arrays horizontally (column-wise)
-> np.repeat()                             : repeats each element a given number of times
-> np.tile()                               : repeats the whole array a given number of times
-> np.r_[a, b]                             : shortcut for joining arrays; same as np.concatenate((arr1, arr2))
-> np.intersect1d(a, b)                    : common elements between two arrays
-> np.union1d(a, b)                        : all unique elements from both arrays
-> np.setdiff1d(a, b)                      : elements in a but not in b
-> np.setxor1d(a, b)                       : elements in either array but not both
-> np.where(condition)                     : returns a tuple of indices where condition is True
-> map with Multiple Iterables             : apply a function across multiple iterables element-by-element
-> zip in python                           : pair elements from multiple iterables together into tuples
-> arr[:, [new_column_order]]              : swap/reorder columns
-> arr[[new_row_order], :]                 : swap/reorder rows
-> arr[::-1, :]                            : reverse rows
-> arr[:, ::-1]                            : reverse columns
-> np.random.uniform                       : random floats in a given range
-> np.set_printoptions(precision=N)        : control decimal display precision
-> np.set_printoptions(suppress=True)      : suppress scientific notation
-> np.set_printoptions(threshold=N)        : limit items shown in output
-> np.genfromtxt()                         : load data from a text/CSV file with flexible options
-> np.mean(), np.median(), np.std()        : basic statistical functions
-> normalization formula                   : (x - min) / (max - min)
-> softmax formula                         : e^x_i / sum(e^x_j) — converts scores into a probability distribution
